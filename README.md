# Currency converter
Simple Currency converter from SGD to 8 most traded currencies
- US dollar (USD)
- Euro (EUR)
- Japanese yen (JPY)
- Pound sterling (GBP)
- Australian dollar (AUD)
- Canadian dollar (CAD)
- Swiss franc (CHF)
- Chinese Yuan (CNY)

Pulls the latest rates from https://open.er-api.com/
